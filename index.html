Fitur Unggulan

Lima mesin.
Satu ekstensi.

CekSINTA dibangun dengan lima engine khusus yang bekerja bersamaan untuk memberikan informasi jurnal paling lengkap.

🔬
FORGE Engine
Deteksi otomatis nama jurnal di Google Scholar dan Garuda, lalu tampilkan badge SINTA S1–S6 dalam waktu kurang dari satu detik.
// core detection
🌐
ATLAS Engine
Menampilkan peringkat Scopus Q1–Q4 beserta SJR Score dan H-index dari database 32.000+ jurnal internasional.
// scopus data
🔖
VAULT — Simpan Jurnal
Simpan jurnal yang sudah dicek ke dalam koleksi pribadi. Export ke CSV kapan saja untuk keperluan daftar pustaka.
// bookmark & export
🦅
RAPTOR Engine
Deteksi jurnal predator secara otomatis menggunakan Beall's List dan database publisher predator lokal Indonesia.
// predatory detection
🔍
Cari Jurnal
Cari jurnal langsung dari popup ekstensi dengan filter peringkat SINTA, Scopus, dan status aktif/tidak aktif.
// search & filter
🌙
Dark Mode + Multi-Platform
Tampilan gelap yang nyaman untuk mata. Bekerja di Google Scholar, Garuda, ResearchGate, dan Semantic Scholar.
// ui & platforms

Kebijakan & Ketentuan

🔒 Kebijakan Privasi

Terakhir diperbarui: 21 April 2026

1. Data yang Dikumpulkan

CekSINTA mengumpulkan data minimal yang diperlukan untuk beroperasi. Secara default, ekstensi hanya membaca nama jurnal dari halaman yang aktif dibuka di browser kamu untuk dikirimkan ke API kami. Tidak ada data identitas pribadi yang dikumpulkan tanpa persetujuan eksplisit.

2. Data Opsional (Login Google)

Jika kamu memilih login dengan Google, kami menyimpan alamat email kamu untuk menghubungkan status berlangganan ke akun. Email tidak dibagikan kepada pihak ketiga manapun dan tidak digunakan untuk tujuan pemasaran tanpa izin.

3. Data Pembayaran

Transaksi pembayaran diproses oleh Midtrans (PT Midtrans). Kami tidak menyimpan informasi kartu atau detail rekening bank kamu. Silakan baca kebijakan privasi Midtrans untuk informasi lebih lanjut.

4. Penyimpanan Lokal

Bookmark jurnal dan preferensi ekstensi disimpan secara lokal di browser kamu menggunakan chrome.storage.local dan tidak dikirimkan ke server kami.

5. Penggunaan Data API

Query nama jurnal yang dikirim ke API kami tidak dikaitkan dengan identitas pengguna dan digunakan semata-mata untuk mengembalikan data akreditasi. Log query dihapus otomatis setelah 30 hari.

6. Hak Pengguna

  • Hak untuk mengakses data yang kami simpan tentang kamu
  • Hak untuk meminta penghapusan data
  • Hak untuk menarik persetujuan kapan saja

Untuk pertanyaan privasi: automeda.app@gmail.com
